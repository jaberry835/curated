Angular RAG System Test Document

This is a test document for the Angular RAG (Retrieval-Augmented Generation) system.

Key Information:
- The system uses Azure OpenAI for embeddings and chat completions
- Documents are stored in Azure Blob Storage
- Text extraction is performed using Azure Document Intelligence
- Vector search is handled by Azure AI Search
- The frontend is built with Angular and uses Angular Material

Technical Details:
- Backend: ASP.NET Core MCP Server
- Frontend: Angular 18 with standalone components
- Authentication: Azure Entra ID (MSAL)
- Storage: Azure Blob Storage
- Search: Azure AI Search with vector indexing
- AI: Azure OpenAI GPT-4 models

Features:
- Document upload and processing
- Automatic text chunking and embedding
- Vector similarity search
- Context injection into LLM prompts
- Real-time chat interface
- Session-based document management
